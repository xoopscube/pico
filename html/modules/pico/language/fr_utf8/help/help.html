Pico Module

Pico is a Trust module, D3 for directory free and duplicable, that can run multiple instances.

The module provides management control in the front-end for static content and publishing workflows.

The module features a simple and granular access permissions assigned to groups and to individual user.

Among other options, custom 'head' with stylesheet and JavaScript by module, category or content.

You can track the history of your document and see past revisions from the front-end, create forms,

override options for each category, forum integration to manage comments, and much more...

Content Management

Categories and content can be moved to other categories or exported to compatible modules.

When Wrap mode is enabled, the virtual path for the content is available.

The following features are available in the backend Category and content management :

  • Name, title
  • Virtual path
  • Weight (order) YSIWYG Editor
  • Visible (public)
  • Show in Navigation
  • Show in Menu
  • Allow comments
  • Bulk Action
  • Granular Permissions
Note the User Interface design is subject to change, but not the logic.
